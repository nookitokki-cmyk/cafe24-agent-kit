# SFTP 접속세팅

## 요청
- OAuth/API 세팅은 하지 않음
- SFTP 접속만 세팅

## 필요한 정보
- 몰 ID: paransky97
- 접속 방식: 일반 운영몰 디자인 SFTP
- 호스트: 로컬 설정 파일 입력 완료
- 포트: 8008
- 아이디: 로컬 설정 파일 입력 완료
- 비밀번호: 로컬 설정 파일 입력 완료, 채팅 공유 금지
- 쓰기 허용 경로: 작업할 skin_code 확인 후 입력

## 설정 파일 위치
- `C:\nookitokki\cafe24-kit-작업본\mcp\config\sftp_paransky97.json`

## 파일 형식
```json
{
  "host": "카페24에서 발급된 호스트",
  "port": 22,
  "username": "카페24에서 발급된 아이디",
  "password": "로컬 파일에만 입력",
  "write_allowed": []
}
```

## 주의
- 호스트와 포트는 예시값으로 추측하지 않고 카페24 발급 화면의 값을 그대로 사용한다.
- `write_allowed`가 빈 배열이면 업로드는 거부되고 읽기만 가능하다.
- 작업할 스킨 폴더가 확인되면 예: `["/skin1"]`처럼 허용 경로를 추가한다.
