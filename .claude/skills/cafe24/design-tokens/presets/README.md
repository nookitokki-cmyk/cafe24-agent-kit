# 디자인 시스템 프리셋 — 골라서 바로 쓰기

> Figma 시안이 없어도 **검증된 디자인 시스템을 골라** 색·폰트만 바꿔 전체 페이지에 적용하는 경로.
> (시안이 있으면 `/레퍼런스인입`으로 가져오기 — 둘 다 지원)

## 프리셋 3종

| 프리셋 | 파일 | 톤 | 포인트 색 | 어울리는 업종 |
|---|---|---|---|---|
| **미니멀 모노** | `minimal.json` | 흑백·넓은 여백·절제 | 없음(모노) | 편집숍·가전·미니멀 패션 |
| **럭셔리 다크** | `luxury.json` | 다크 배경·골드·세리프 | 골드 `#c8a97e` | 주얼리·뷰티·프리미엄 |
| **소프트 데일리** | `soft.json` | 따뜻한 크림·둥근 모서리 | 테라코타 `#b8623d` | 리빙·데일리·셀렉트샵 |

## 쓰는 법 (3줄)
1. 위에서 하나 고른다 → 해당 `.json`을 이 클라이언트 토큰으로 복사(`clients/{몰}/04_design/tokens.json`)
2. **색·폰트만** 내 브랜드에 맞게 바꾼다 (구조는 그대로)
3. `/토대정리`가 이 토큰을 `:root` CSS 변수로 세팅 → `09-full-renewal` 워크플로우가 전 페이지에 적용

## 형식
- 모든 프리셋은 `tokens.schema.json` 준수 (colors·typography·spacing·radius·shadow·breakpoints).
- 브레이크포인트 375·768·1280·1440, `nk-` 접두사, `font-style:italic` 금지 — 누끼토끼 룰 반영.
- 폰트: 기본 Pretendard / 럭셔리는 Bricolage Grotesque(본문)+Marcellus(영문 디스플레이).

> 프리셋은 **출발점**이다. 색·간격을 브랜드에 맞게 조정해 완성한다. (`builder-guide.md` 참고)
