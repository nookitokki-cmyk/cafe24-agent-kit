# 메인 히어로 — 쇼핑몰 첫 화면 (배너 + 신상품 + 베스트)

## 이게 뭔가요?
쇼핑몰 메인 페이지 맨 위에 보이는 큰 배너 이미지와, 그 아래에 줄지어 나오는 신상품·베스트 상품 목록을 한 번에 만드는 레시피입니다.
손님이 사이트에 처음 들어왔을 때 가장 먼저 보이는 핵심 화면이에요.
배너는 슬라이드로 넘기거나 고정 이미지로 쓸 수 있습니다.

## 어떤 페이지에서 쓰나요?
- 쇼핑몰 메인 페이지(`index.html`)에서만 사용합니다.
- "첫 화면에 이벤트 배너 + 신상품 8개 + 베스트 8개 보여줘"를 구현할 때 딱 맞는 레시피입니다.

## 필요한 모듈 (순서대로)

| 모듈 ID | 역할 | 필수여부 |
|---------|------|---------|
| `ModuleBannerSmall` | 메인 배너(이미지 슬라이드 또는 고정) | 필수 |
| `ModuleProductNew` | 신상품 목록 자동 출력 | 필수 |
| `ModuleProductBest` | 베스트 상품 목록 자동 출력 | 선택 |

> **모듈이 뭔가요?** 카페24 스마트디자인에서 상품 목록, 배너, 회원정보 같은 동적 데이터를 자동으로 불러오는 "레고 블록"이라고 생각하면 됩니다.
> HTML 파일 안에 `<module>` 태그를 넣으면 카페24가 자동으로 실제 데이터를 채워줍니다.

## 복사해서 쓰는 HTML

```html
<!--@@--
페이지 타입: 메인 (index.html)
레시피: 01_메인-히어로
--@@-->

<!-- ① 메인 배너 영역 -->
<section class="nk-hero">
  <div class="nk-hero__banner">
    <module name="ModuleBannerSmall" />
  </div>
</section>

<!-- ② 신상품 섹션 -->
<section class="nk-new-products">
  <div class="nk-section-header">
    <h2 class="nk-section-title">NEW ARRIVALS</h2>
    <a href="/product/list.html?cate_no=1" class="nk-view-all">전체보기</a>
  </div>
  <div class="nk-product-grid">
    <module name="ModuleProductNew">
      <div class="nk-product-item">
        <!--상품 이미지-->
        <a href="{$product_url}">
          <div class="nk-product-thumb">
            <img src="{$thumbnail_url}" alt="{$product_name}" loading="lazy">
          </div>
        </a>
        <!--상품 정보-->
        <div class="nk-product-info">
          <p class="nk-product-name">{$product_name}</p>
          <p class="nk-product-price">
            <span class="nk-price-sale">{$price|number_format}원</span>
          </p>
        </div>
      </div>
    </module>
  </div>
</section>

<!-- ③ 베스트 상품 섹션 -->
<section class="nk-best-products">
  <div class="nk-section-header">
    <h2 class="nk-section-title">BEST</h2>
    <a href="/product/list.html?cate_no=1" class="nk-view-all">전체보기</a>
  </div>
  <div class="nk-product-grid">
    <module name="ModuleProductBest">
      <div class="nk-product-item">
        <!--상품 이미지-->
        <a href="{$product_url}">
          <div class="nk-product-thumb">
            <img src="{$thumbnail_url}" alt="{$product_name}" loading="lazy">
          </div>
        </a>
        <!--상품 정보-->
        <div class="nk-product-info">
          <p class="nk-product-name">{$product_name}</p>
          <p class="nk-product-price">
            <span class="nk-price-sale">{$price|number_format}원</span>
          </p>
        </div>
      </div>
    </module>
  </div>
</section>
```

## 여기만 바꾸세요

| 바꿀 부분 | 현재 값 | 설명 |
|-----------|---------|------|
| `"전체보기"` 링크 | `/product/list.html?cate_no=1` | 카페24 관리자 → 상품 → 카테고리에서 번호 확인 후 교체 |
| `"NEW ARRIVALS"` 텍스트 | `NEW ARRIVALS` | 원하는 섹션 제목으로 변경 |
| `"BEST"` 텍스트 | `BEST` | 원하는 섹션 제목으로 변경 |
| 그리드 열 수 | `repeat(4, 1fr)` | 3열로 바꾸려면 `repeat(3, 1fr)` |
| 섹션 간격 | `margin: 60px auto` | 숫자를 키우면 위아래 여백이 넓어짐 |

> **배너 이미지/슬라이드 설정은 어디서?**
> 카페24 관리자 → [디자인] → [배너관리]에서 `ModuleBannerSmall`에 연결된 배너를 등록·수정합니다. HTML 코드를 건드릴 필요 없습니다.

## 자주 하는 실수

1. **신상품이 안 나와요**
   → `ModuleProductNew`는 상품 등록 날짜 기준으로 자동 출력됩니다. 관리자에서 상품의 "전시 여부"가 **전시함**으로 설정되어 있는지 확인하세요.

2. **모바일에서 4열이 그대로 나와요**
   → CSS의 `@media (max-width: 768px)` 블록이 누락됐거나, 스킨 기본 CSS와 충돌 중입니다. 공통 CSS 파일 맨 아래에 위 모바일 코드를 추가하세요.
