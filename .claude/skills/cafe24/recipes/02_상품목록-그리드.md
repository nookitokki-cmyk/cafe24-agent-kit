# 상품목록 그리드 — 카테고리 페이지 상품 목록

## 이게 뭔가요?
"여성 상의", "신발", "베스트" 같은 카테고리를 클릭했을 때 나오는 상품 목록 페이지입니다.
상품들이 격자(그리드) 형태로 줄지어 나오고, 페이지 아래에 1·2·3·4 페이지 번호도 나옵니다.
카테고리 이름, 상품 수, 정렬 기준(최신순/인기순 등)도 함께 표시됩니다.

## 어떤 페이지에서 쓰나요?
- 카테고리 상품 목록 페이지(`product_list.html`)에서 사용합니다.
- "카테고리 클릭하면 상품 목록이 4열 그리드로 나오게 해줘"를 구현할 때 이 레시피를 씁니다.

## 필요한 모듈 (순서대로)

| 모듈 ID | 역할 | 필수여부 |
|---------|------|---------|
| `product_headcategory` | 상단 카테고리 이름(타이틀) 표시 | 필수 |
| `product_displaycategory` | 현재 카테고리의 하위 카테고리 탭 | 선택 |
| `product_normalmenu` | 정렬 드롭다운(최신순/판매순 등) + 상품 수 표시 | 필수 |
| `product_list` | 실제 상품 목록 (썸네일·이름·가격) | 필수 |
| `ModulePagination` | 하단 페이지 번호 버튼 | 필수 |

> **핵심 파라미터 미리 알기**
> - `display_count` : 한 페이지에 보여줄 상품 수 (예: 16, 20, 24)
> - `column` : 한 줄에 몇 개씩 나열할지 (예: 4 → 4열 그리드)
> 이 두 숫자만 바꾸면 레이아웃이 달라집니다.

## 복사해서 쓰는 HTML

```html
<!--@@--
페이지 타입: 상품 목록 (product_list.html)
레시피: 02_상품목록-그리드
--@@-->

<!-- ① 카테고리 이름 (페이지 상단 타이틀) -->
<div class="nk-category-header">
  <module name="product_headcategory">
    <h1 class="nk-category-title">{$category_name}</h1>
  </module>
</div>

<!-- ② 하위 카테고리 탭 (있을 때만 노출) -->
<div class="nk-subcategory-tabs">
  <module name="product_displaycategory">
    <ul class="nk-tab-list">
      <li class="nk-tab-item {$selected}">
        <a href="{$category_url}">{$category_name}</a>
      </li>
    </ul>
  </module>
</div>

<!-- ③ 정렬 + 상품 수 표시 -->
<div class="nk-list-toolbar">
  <module name="product_normalmenu">
    <span class="nk-product-count">
      총 <strong>{$total_count}</strong>개 상품
    </span>
    <select class="nk-sort-select" onchange="location.href=this.value">
      <option value="{$sort_url_date}" {$date_selected}>최신순</option>
      <option value="{$sort_url_best}" {$best_selected}>인기순</option>
      <option value="{$sort_url_price_asc}" {$price_asc_selected}>낮은가격순</option>
      <option value="{$sort_url_price_desc}" {$price_desc_selected}>높은가격순</option>
    </select>
  </module>
</div>

<!-- ④ 상품 목록 그리드 (핵심) -->
<div class="nk-product-list">
  <module name="product_list" display_count="16" column="4">
    <!-- 상품 없을 때 -->
    <div module="empty">
      <p class="nk-empty-msg">등록된 상품이 없습니다.</p>
    </div>
    <!-- 상품 있을 때 -->
    <div module="list">
      <ul class="nk-product-grid">
        <li module="item" class="nk-product-item">
          <a href="{$product_url}">
            <!-- 상품 이미지 -->
            <div class="nk-product-thumb">
              <img src="{$thumbnail_url}" alt="{$product_name}" loading="lazy">
              <!-- 품절 뱃지 (품절일 때만 나타남) -->
              <span class="nk-badge-soldout" {$soldout|display}>품절</span>
            </div>
            <!-- 상품 정보 -->
            <div class="nk-product-info">
              <p class="nk-product-brand">{$brand_name}</p>
              <p class="nk-product-name">{$product_name}</p>
              <p class="nk-product-price">
                <!-- 할인가가 있을 때 -->
                <span class="nk-price-original {$sale_price|display}">{$price|number_format}원</span>
                <!-- 최종 판매가 -->
                <span class="nk-price-sale">{$sale_price|number_format}원</span>
              </p>
            </div>
          </a>
        </li>
      </ul>
    </div>
  </module>
</div>

<!-- ⑤ 페이지 번호 -->
<div class="nk-pagination">
  <module name="ModulePagination" />
</div>
```

## 여기만 바꾸세요

| 바꿀 부분 | 현재 값 | 설명 |
|-----------|---------|------|
| `display_count` | `16` | 한 페이지에 보여줄 상품 수. 20, 24, 32 등 |
| `column` | `4` | 한 줄에 몇 개씩. 3이나 4가 일반적 |
| 그리드 CSS 열 수 | `repeat(4, 1fr)` | `column` 값과 맞춰서 변경 |
| 정렬 기준 텍스트 | `최신순`, `인기순` 등 | 원하는 이름으로 수정 가능 |

> **`display_count`와 `column`은 짝을 맞춰야 합니다.**
> 예: `display_count="12"` + `column="3"` → 한 페이지에 4줄 × 3열 = 12개

## 자주 하는 실수

1. **`column` 숫자를 바꿨는데 실제 화면은 안 바뀌어요**
   → HTML의 `column="4"` 속성을 바꿔도 CSS의 `grid-template-columns: repeat(4, 1fr)` 도 함께 바꿔야 합니다. 두 군데 모두 수정하세요.

2. **하위 카테고리 탭이 이상하게 나와요**
   → `product_displaycategory` 모듈은 현재 카테고리에 하위 카테고리가 실제로 등록되어 있어야 작동합니다. 관리자 → [상품] → [카테고리 관리]에서 하위 카테고리를 추가해 주세요.
