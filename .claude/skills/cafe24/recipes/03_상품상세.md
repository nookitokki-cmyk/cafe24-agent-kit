# 상품 상세 — 이미지·옵션·구매버튼·후기 한 페이지에

## 이게 뭔가요?
상품을 클릭했을 때 나오는 상세 페이지 전체입니다.
왼쪽엔 상품 이미지, 오른쪽엔 이름·가격·옵션·구매 버튼이 있고,
아래로 내려가면 상품 상세 설명, 다른 추천 상품, 구매 후기, Q&A가 이어서 나옵니다.
쇼핑몰에서 "이 페이지 하나가 구매를 결정"하는 가장 중요한 페이지입니다.

## 어떤 페이지에서 쓰나요?
- 상품 상세 페이지(`product_detail.html`)에서 사용합니다.
- 상품을 클릭했을 때 열리는 페이지 전체를 구성할 때 이 레시피를 씁니다.

## 필요한 모듈 (순서대로)

| 모듈 ID | 역할 | 필수여부 |
|---------|------|---------|
| `product_image` | 상품 메인 이미지 + 썸네일 목록 | 필수 |
| `product_detail` | 상품 이름·가격·요약 설명 | 필수 |
| `product_option` | 색상·사이즈 등 옵션 선택 UI | 선택 |
| `product_action` | 구매하기·장바구니·찜하기 버튼 | 필수 |
| `product_detaildesign` | 상세 설명 이미지(에디터 입력 내용) | 필수 |
| `product_relation` | 연관 상품(함께 보면 좋은 상품) | 선택 |
| `product_review` | 구매 후기 탭 | 선택 |
| `product_qna` | 상품 문의 탭 | 선택 |

## 복사해서 쓰는 HTML

```html
<!--@@--
페이지 타입: 상품 상세 (product_detail.html)
레시피: 03_상품상세
--@@-->

<!-- 상단: 이미지 + 구매 정보 (좌우 2단 레이아웃) -->
<div class="nk-detail-wrap">

  <!-- ① 상품 이미지 영역 (왼쪽) -->
  <div class="nk-detail-gallery">
    <module name="product_image">
      <!-- 메인 이미지 -->
      <div class="nk-main-image">
        <img src="{$main_image_url}" alt="{$product_name}" id="nkMainImg">
      </div>
      <!-- 썸네일 목록 -->
      <div class="nk-thumb-list">
        <ul>
          <li module="thumbnail">
            <img src="{$thumbnail_url}" alt="" class="nk-thumb-item"
              onclick="document.getElementById('nkMainImg').src='{$main_image_url}'">
          </li>
        </ul>
      </div>
    </module>
  </div>

  <!-- ② 상품 기본 정보 + 옵션 + 구매버튼 (오른쪽) -->
  <div class="nk-detail-info">

    <!-- 상품 이름·가격 -->
    <module name="product_detail">
      <p class="nk-detail-brand">{$brand_name}</p>
      <h1 class="nk-detail-name">{$product_name}</h1>
      <div class="nk-detail-price">
        <span class="nk-price-original {$sale_price|display}">{$price|number_format}원</span>
        <span class="nk-price-final">{$sale_price|number_format}원</span>
      </div>
      <!-- 배송비 -->
      <div class="nk-shipping-info">
        <span class="nk-shipping-label">배송비</span>
        <span class="nk-shipping-value">{$shipping_fee}</span>
      </div>
      <!-- 상품 요약 정보 (옵션 없을 때 여기에 수량 표시) -->
      <div class="nk-detail-summary">{$summary_description}</div>
    </module>

    <!-- ③ 옵션 선택 -->
    <div class="nk-option-area">
      <module name="product_option">
        <!-- 옵션이 있을 때 -->
        <div module="option">
          <div class="nk-option-row">
            <label class="nk-option-label">{$option_name}</label>
            <!-- 드롭다운 방식 -->
            <select name="product_option_{$option_no}" class="nk-option-select">
              <option value="">-- {$option_name} 선택 --</option>
              <li module="option_item">
                <option value="{$option_detail_no}" {$disabled}>{$option_detail_name}
                  {$option_price|display( (+{$option_price|number_format}원))}
                </option>
              </li>
            </select>
          </div>
        </div>
        <!-- 선택된 옵션 목록 + 수량 -->
        <div module="selected" class="nk-selected-options">
          <ul>
            <li module="selected_item" class="nk-selected-item">
              <span class="nk-selected-name">{$product_name} / {$option_name}</span>
              <div class="nk-qty-control">
                <button type="button" onclick="changeQty('{$basket_product_no}', -1)">-</button>
                <input type="text" value="{$quantity}" id="quantity_{$basket_product_no}"
                  class="nk-qty-input" readonly>
                <button type="button" onclick="changeQty('{$basket_product_no}', 1)">+</button>
              </div>
              <span class="nk-selected-price">{$product_price|number_format}원</span>
              <button type="button" class="nk-remove-option"
                onclick="removeSelectedOption('{$basket_product_no}')">×</button>
            </li>
          </ul>
          <!-- 합계 금액 -->
          <div class="nk-total-price">
            <span>합계</span>
            <strong id="totalPrice">{$total_price|number_format}원</strong>
          </div>
        </div>
      </module>
    </div>

    <!-- ④ 구매 버튼 -->
    <div class="nk-action-buttons">
      <module name="product_action">
        <!-- 장바구니 -->
        <button type="button" class="nk-btn-cart" id="btnCart">장바구니</button>
        <!-- 바로 구매 -->
        <button type="button" class="nk-btn-buy" id="btnBuy">바로 구매</button>
        <!-- 찜하기 -->
        <button type="button" class="nk-btn-wish {$wish|display(on)}" id="btnWish">
          <ph-icon name="heart" class="nk-wish-icon"></ph-icon>
        </button>
      </module>
    </div>

  </div><!-- /nk-detail-info -->
</div><!-- /nk-detail-wrap -->

<!-- ⑤ 탭 메뉴 (상세설명 / 후기 / Q&A) -->
<div class="nk-detail-tabs">
  <ul class="nk-tab-nav" role="tablist">
    <li><a href="#nk-tab-desc" class="nk-tab-link active">상품 상세</a></li>
    <li><a href="#nk-tab-review" class="nk-tab-link">후기</a></li>
    <li><a href="#nk-tab-qna" class="nk-tab-link">문의</a></li>
  </ul>
</div>

<!-- ⑥ 상세 설명 (에디터에서 입력한 이미지/텍스트) -->
<div id="nk-tab-desc" class="nk-tab-content">
  <module name="product_detaildesign">
    <div class="nk-detail-description">{$detail_description}</div>
  </module>
</div>

<!-- ⑦ 연관 상품 -->
<div class="nk-related-products">
  <h3 class="nk-section-title">함께 보면 좋은 상품</h3>
  <module name="product_relation">
    <ul class="nk-product-grid">
      <li module="item" class="nk-product-item">
        <a href="{$product_url}">
          <div class="nk-product-thumb">
            <img src="{$thumbnail_url}" alt="{$product_name}" loading="lazy">
          </div>
          <p class="nk-product-name">{$product_name}</p>
          <p class="nk-price-sale">{$price|number_format}원</p>
        </a>
      </li>
    </ul>
  </module>
</div>

<!-- ⑧ 구매 후기 탭 -->
<div id="nk-tab-review" class="nk-tab-content">
  <module name="product_review" />
</div>

<!-- ⑨ 상품 문의 탭 -->
<div id="nk-tab-qna" class="nk-tab-content">
  <module name="product_qna" />
</div>
```

## 여기만 바꾸세요

| 바꿀 부분 | 현재 값 | 설명 |
|-----------|---------|------|
| 탭 이름 | `상품 상세`, `후기`, `문의` | 원하는 이름으로 수정 가능 |
| `함께 보면 좋은 상품` | 텍스트 | 연관 상품 섹션 제목 변경 |
| 구매 버튼 텍스트 | `장바구니`, `바로 구매` | 원하는 문구로 교체 |
| `nk-btn-buy` 배경색 | `#111` | 브랜드 컬러로 변경 |

## 자주 하는 실수

1. **옵션을 선택했는데 합계 금액이 안 바뀌어요**
   → `product_option` 모듈의 JavaScript 동작은 카페24가 자동으로 처리합니다. 단, 스킨 공통 JS 파일(`common.js` 또는 `basic.js`)이 페이지에 포함되어 있어야 합니다. 누락 여부를 확인하세요.

2. **상세 설명(에디터 내용)이 너무 작게 나와요**
   → `product_detaildesign` 모듈이 출력하는 내용은 관리자 에디터에서 입력한 HTML 그대로입니다. CSS `max-width`를 `nk-detail-description` 클래스에 별도 지정하거나, 에디터에서 이미지 너비를 100%로 설정하세요.
