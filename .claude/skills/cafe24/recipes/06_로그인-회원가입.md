# 로그인·회원가입 — 로그인 폼 + 회원가입 폼

## 이게 뭔가요?
고객이 쇼핑몰에 로그인하거나 처음 가입할 때 쓰는 페이지입니다.
로그인 페이지에서는 아이디·비밀번호 입력창과 로그인 버튼이 나오고,
회원가입 페이지에서는 이름·이메일·비밀번호 등 정보를 입력하는 폼이 나옵니다.
소셜 로그인(카카오·네이버 등)과 비회원 주문 분기도 이 페이지에서 처리됩니다.

## 어떤 페이지에서 쓰나요?
- 로그인 페이지: `member_login.html`
- 회원가입 페이지: `member_join.html`

## 필요한 모듈 (순서대로)

| 모듈 ID | 역할 | 필수여부 |
|---------|------|---------|
| `member_login` | 아이디·비밀번호 입력 + 로그인 버튼 + 소셜로그인 | 필수 |
| `member_join` | 이름·이메일·비밀번호 등 가입 폼 | 필수 |

## 꼭 알아야 할 개념

### returnUrl이란?
로그인 성공 후 어느 페이지로 이동할지를 결정하는 주소입니다.
예를 들어 장바구니에서 로그인 페이지로 넘어왔다면, 로그인 완료 후 다시 장바구니로 돌아가야 합니다.
카페24가 URL 뒤에 `?returnUrl=/order/basket.html`처럼 자동으로 붙여주기 때문에 HTML에서 직접 건드릴 필요는 없습니다.

### 비회원 주문 분기란?
일부 쇼핑몰은 가입 없이 주문할 수 있게 허용합니다.
이 경우 로그인 페이지에 "비회원 주문" 버튼을 함께 보여줍니다.
카페24 관리자 → [회원] → [회원 정책]에서 허용 여부를 설정하면, 모듈이 자동으로 버튼을 표시/숨김 처리합니다.

### [A] 로그인 페이지 (`member_login.html`)

```html
<!--@@--
페이지 타입: 로그인 (member_login.html)
레시피: 06_로그인-회원가입 (A)
--@@-->

<div class="nk-login-wrap">
  <h1 class="nk-page-title">로그인</h1>

  <module name="member_login">

    <!-- ① 일반 로그인 폼 -->
    <div class="nk-login-box">
      <form name="loginForm" method="post">
        <!-- 아이디 -->
        <div class="nk-form-group">
          <label class="nk-form-label" for="member_id">아이디</label>
          <input type="text" id="member_id" name="member_id"
            class="nk-form-input" placeholder="아이디를 입력하세요"
            value="{$member_id}" autocomplete="username">
        </div>
        <!-- 비밀번호 -->
        <div class="nk-form-group">
          <label class="nk-form-label" for="member_passwd">비밀번호</label>
          <input type="password" id="member_passwd" name="member_passwd"
            class="nk-form-input" placeholder="비밀번호를 입력하세요"
            autocomplete="current-password">
        </div>

        <!-- 아이디 저장 체크박스 -->
        <div class="nk-login-options">
          <label class="nk-check-label">
            <input type="checkbox" name="save_id" {$save_id_checked}>
            아이디 저장
          </label>
          <!-- 자동 로그인 -->
          <label class="nk-check-label">
            <input type="checkbox" name="auto_login" {$auto_login_checked}>
            자동 로그인
          </label>
        </div>

        <!-- 로그인 버튼 -->
        <button type="submit" class="nk-btn-login">로그인</button>

        <!-- 아이디/비밀번호 찾기 + 회원가입 링크 -->
        <div class="nk-login-links">
          <a href="/member/findid.html" class="nk-link">아이디 찾기</a>
          <span class="nk-divider">|</span>
          <a href="/member/findpw.html" class="nk-link">비밀번호 찾기</a>
          <span class="nk-divider">|</span>
          <a href="/member/join.html" class="nk-link">회원가입</a>
        </div>

      </form>
    </div>

    <!-- ② 소셜 로그인 (카카오·네이버 등 — 관리자에서 설정한 것만 나타남) -->
    <div class="nk-social-login">
      <p class="nk-divider-text">또는 소셜 계정으로 로그인</p>
      <div module="sns_login">
        <!-- 카카오 -->
        <a href="{$kakao_login_url}" class="nk-btn-social nk-social-kakao"
          {$kakao_login|display}>
          카카오로 로그인
        </a>
        <!-- 네이버 -->
        <a href="{$naver_login_url}" class="nk-btn-social nk-social-naver"
          {$naver_login|display}>
          네이버로 로그인
        </a>
      </div>
    </div>

    <!-- ③ 비회원 주문 버튼 (관리자에서 비회원 주문 허용 시만 노출) -->
    <div class="nk-guest-order {$guest_order|display}">
      <p class="nk-guest-desc">회원가입 없이도 주문할 수 있습니다.</p>
      <a href="{$guest_order_url}" class="nk-btn-guest">비회원 주문</a>
    </div>

  </module>

</div>
```

## 여기만 바꾸세요

| 바꿀 부분 | 현재 값 | 설명 |
|-----------|---------|------|
| 배송 메모 선택지 | (없음) | 로그인 페이지엔 없음. 주문서 레시피 참고 |
| `nk-btn-login` 배경색 | `#111` | 브랜드 컬러로 변경 |
| `nk-btn-join` 배경색 | `#111` | 브랜드 컬러로 변경 |
| 소셜 로그인 버튼 색상 | 카카오 `#FEE500` / 네이버 `#03C75A` | 브랜드 공식 컬러이므로 변경 비권장 |
| 약관 항목 수 | 이용약관 + 개인정보 + 마케팅 | 필요 없는 항목은 삭제 가능 |

## 자주 하는 실수

1. **소셜 로그인 버튼이 안 나타나요**
   → `{$kakao_login|display}`는 관리자 → [회원] → [소셜 로그인 설정]에서 카카오 연동이 완료된 경우에만 자동으로 나타납니다. 연동 설정을 먼저 진행하세요.

2. **가입 후 메인으로 이동하지 않고 오류가 나요**
   → `member_join` 모듈의 폼 제출은 카페24 내부 로직으로 처리됩니다. `onsubmit="return validateJoin()"` 부분의 `validateJoin` 함수가 스킨 JS에 없으면 오류가 납니다. 이 부분을 `onsubmit`을 제거하거나 카페24 기본 `join.js`를 포함시키세요.
