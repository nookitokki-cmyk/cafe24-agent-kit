사용자가 `/캐시확인` 을 요청했습니다.

## 체크리스트 (순서대로)

### 1. 올린 파일이 맞는가

「SFTP로 수정한 **경로**와 라이브 URL이 같은 스킨인가요?」

- 파트너: `/sde_design/base/` vs `/mobile/`
- 일반몰: `/skin14/` 등 — `/접속세팅` skin 표 재확인

### 2. 서버 캐시 대기

「업로드 후 **2~5분** 기다렸나요?」 ⚠️ F3

### 3. 브라우저 캐시

「URL 끝에 **`?v=2`** (숫자 증가) 붙여 하드 리로드 했나요?」

- `@css` 링크에 `?v=` 붙이면 **안 됨** ⚠️ F4 — 페이지 URL에만

### 4. 라이브 vs 관리자 미리보기

「**실제 쇼핑몰 URL**로 보고 있나요? (관리자 미리보기만 X)」

### 5. MOBILE_WEB

「PC만 바뀌고 모바일이 `/m/` 이면 관리자 **모바일 디자인 사용 안 함** 확인」

## 완료

```
[캐시확인]
- 권장: ?v=N 리로드 · 3분 대기 · 스킨 경로 재확인
- 계속 같으면: /요소측정 으로 라이브 CSS 값 읽기
```
