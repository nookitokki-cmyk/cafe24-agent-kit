# 설치·사용 안내 (카페24 에이전트 키트)

> 이 키트는 **카페24 스킨을 AI(Cursor·Claude Code)에게 시켜서 안 깨지게 만드는 작업 도구상자**입니다.
> 코드를 몰라도 됩니다. 아래 순서대로 따라 하시면 됩니다.
> **경로는 사람마다 다릅니다** — 이 키트를 **아무 폴더에나** 두고, 그 폴더를 IDE로 열기만 하면 됩니다. (고정 경로를 외울 필요 없음)

---

## 0. 준비물
- **Python 3.10 이상** — 터미널에서 `py -3 --version` (Mac/Linux는 `python3 --version`)
- **Cursor** 또는 **Claude Code**
- **내 몰 ID** — 쇼핑몰 주소 `https://○○○.cafe24.com` 의 `○○○` 부분

---

## 1. 키트 받기 (셋 중 하나)
1. **GitHub Release** → `cafe24-agent-kit-v*.zip` 다운로드 → **원하는 폴더에 압축 해제**
2. **이미 옛 버전이 있으면** → `py -3 cli.py kit-update --from-github`
3. **(권한 있으면)** `git clone`

---

## 2. 설치 (처음 한 번만)

압축 푼 폴더를 **IDE로 연 뒤**, 하단 터미널에서:

```bash
cd mcp
py -3 -m pip install -r requirements.txt     # 1) 패키지 설치
py -3 -c "import server"                       # 2) 에러 없이 넘어가면 정상
py -3 cli.py scaffold --mall 내몰ID            # 3) 내 몰 설정 파일 생성
```

4) **IDE에 MCP 연결**
   - `.cursor/mcp.json.example` 를 복사해 같은 폴더에 **`mcp.json`** 으로 이름 변경
   - IDE 재시작 → MCP 목록에 **`cafe24-mcp`** 초록불 확인
   - (경로는 `${workspaceFolder}` 로 자동 처리 — 어디에 두든 동작)

5) **OAuth 연결** — 채팅에 **`/API발급`** 또는:
```bash
py -3 cli.py auth-url --mall 내몰ID
py -3 cli.py code "oauth-callback?code=..." --mall 내몰ID
py -3 cli.py status --mall 내몰ID
```

6) **확인** — `py -3 smoke_test.py` → config 없이도 5/9면 정상

> Windows는 `py -3`, Mac/Linux는 `python3`. 상세·트러블슈팅: `agent-kit/00_시작하기/05b-MCP-등록.md`

---

## 3. 사용 (매번)

IDE 채팅에서:

| 하고 싶은 것 | 입력 |
|---|---|
| 뭐부터 할지 모를 때 | **`/도움말`** |
| 접속 정보 세팅 | **`/접속세팅`** |
| 수정 전 실측 | **`/요소측정`** |
| 디자인 수정 시키기 | **`/디자인수정`** + 자연어로 지시 |
| **새 클라이언트 시작** | **`/새클라이언트`** 또는 그냥 "새 클라이언트 시작할게" |

**처음이라면** 이 실습부터: `agent-kit/00_시작하기/첫수정-1건-성공.md`
**막혔을 때:** `agent-kit/00_시작하기/실패복구-가이드.md`

---

## 4. 새 클라이언트(몰) 추가

채팅에 **`/새클라이언트`** 라고 하거나 "새 클라이언트 시작할게" 라고만 해도 됩니다.
**몰 ID 하나만 물어본 뒤**, `agent-kit/clients/{몰ID}/` 폴더가 자동으로 만들어집니다(요청·수정·레퍼런스·작업로그 칸 포함). 클라이언트는 이 폴더 아래 **계속 누적**됩니다.

> 클라이언트 폴더의 실제 정보·비밀번호는 **커밋·배포하지 마세요** (자동으로 git에서 제외됩니다).
