## 가장 쉬운 길 (권장 — 핑퐁식)

1. **키트 받기** — GitHub Release에서 `cafe24-agent-kit.zip` 다운로드 → **아무 폴더에나** 압축 해제
   - (이미 옛 버전이 있으면 채팅에 `/키트시작` 대신 `kit-update --from-github` 먼저)
2. 압축 푼 **그 폴더를 Cursor/Claude Code로 열기**
3. 채팅에 **`/키트시작`** → 순서대로 물어봅니다:
   워크스페이스 확인 → 패키지 설치 → 점검 → MCP 연결 → 점검(smoke)
4. 이어서 **`/API발급`** → 카페24 연결 토큰 (이것도 하나씩 물어봄)
5. 끝! 이제 작업 시작

> 준비물: **Python 3.10+**, **Cursor 또는 Claude Code**, **내 몰 ID**(`https://○○○.cafe24.com`의 `○○○`)

## (참고) 손으로 직접 설치하고 싶을 때만

> 위 `/키트시작`이 아래를 대신 안내해 줍니다. 직접 하고 싶은 분만 보세요.
> Windows는 `py -3`, Mac/Linux는 `python3`.

```bash
cd mcp
py -3 -m pip install -r requirements.txt     # 패키지 설치
py -3 -c "import server"                       # 점검(에러 없으면 OK)
py -3 cli.py scaffold --mall 내몰ID            # 내 몰 폴더·설정 생성
```
- `.cursor/mcp.json.example`(Windows) 또는 `.cursor/mcp.json.mac.example`(Mac) → `.cursor/mcp.json` 복사 후 IDE 재시작 (경로는 `${workspaceFolder}`로 자동)
- 토큰: `py -3 cli.py auth-url --mall 내몰ID` → 브라우저 허용 → `py -3 cli.py code "oauth-callback?code=..." --mall 내몰ID`
- 점검: `py -3 smoke_test.py` (config 없이도 5/9면 정상)
