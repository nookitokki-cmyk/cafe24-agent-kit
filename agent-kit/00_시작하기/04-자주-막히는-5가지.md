# 04. 자주 막히는 5가지

> `02_막혔을때/함정-INDEX.md` 와 연결됩니다. 번호는 00_시작하기용 요약입니다.

---

## 함정 #1 — 캐시 (바꿨는데 화면이 안 바뀜)

**증상:** 파일 올렸는데 똑같아 보임.

**처방:**
- 라이브 URL 끝에 `?v=2` 처럼 **숫자 붙여서** 새로고침
- 2~5분 기다리기 (서버 쪽 묶음 지연 ⚠️ F3)

---

## 함정 #2 — `{$변수}` 값이 안 나옴(빈 값)

**증상:** `{$product_name}` 자리가 **빈 값**으로 나옴 — 글자 그대로가 아니라 **아무것도 안 보임**. (라이브 검증 2026-07-06: 스코프 밖 변수는 글자 그대로가 아니라 빈 값)

**처방:** 변수는 **`module="..."` 안**에만. ✅ [basic 가이드](https://developers.cafe24.com/design/front/smart/sdsupport/basic)

---

## 함정 #3 — `@css` 뒤에 `?v=1` 붙임

**증상:** CSS 통째로 안 먹음.

**처방:** `@css`에는 쿼리 붙이지 말 것. 캐시는 **페이지 URL**에 `?v=N`. ⚠️ F4

---

## 함정 #4 — 상품이 1개만 나옴

**증상:** 메인 진열에 상품 하나.

**처방:** `anchorBoxId` 블록 **2개 이상** 나란히 있는지 확인. ⚠️ 실무 (공식 동일 문장 없음)

---

## 함정 #5 — PC만 되고 모바일은 깨짐 (또는 반대)

**증상:** 한쪽만 정상.

**처방:**
- PC 스킨과 **모바일 스킨은 별도**일 수 있음
- `GET /themes?type=mobile` 로 모바일 목록 확인 ✅ API
- SFTP `/mobile` 폴더 ⚠️ 실무
