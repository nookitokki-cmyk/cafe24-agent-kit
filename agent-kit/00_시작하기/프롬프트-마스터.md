### D. 작업 (HTML/EZ)
14. 이 몰 HTML이야. 메인을 `{레퍼런스}`처럼 재설계해 `skin{N}`에 주입 — 모듈 바인딩 유지, `#nk-skinN` 스코프.
15. 이 몰 EZ야. 07/08로 EZ 조각 선별이식→재설계 주입→`data-ez` strip 단계별로.
16. base 함정 자동 차단 — `body min-width`, `.xans-layout-footer` float, `#header:before` 가상요소선 전부 `#nk-skinN`으로.

18. 폰트 Pretendard(+Marcellus·Bricolage), 아이콘 Phosphor, italic·인라인 style 금지로 통일.

### E. 검증·QA
19. 방금 작업 코드리뷰 — `nk-` 접두사·스코프·`!important` 사유·토큰화·모듈 바인딩(blocking 0).
20. 3축 게이트 검증 — visual+override+rule 전부 통과까지.
21. qa-loop 돌려줘 — 합격 점수까지 자동 수정·재검증 반복.

### F. 배포·운영
22. `skin{N}` 비활성이지? 덮기 전 백업하고, 올린 뒤 미리보기(`?skin=skin{N}`)로 캡처해줘.

## PART 2 — 정밀 분석·구현 (대상·작업·산출 구조)

### 25. base CSS 전수 분석 (함정 지도)
대상: `~/Downloads/{백업}/skin{N}/layout/basic/css/*.css` + `css/module/**/*.css`
작업: nk 전역 셀렉터(body·#header·#footer·#wrap·#contents·a·ul·.xans-* 등)에 base가 거는 규칙 전수 추출. 가상요소(:before/:after)·!important·고정폭·float·border 포함.
산출: "파일:줄 → 셀렉터 → 속성 → 충돌?(상/중/하) → 처방(#nk-skinN 1줄)" 표.
이걸로: `nk-ez-override.css` 일괄 차단 + 구현 가이드. (추측 금지, 줄번호 정확히)

### 26. 레퍼런스 → 토큰·에셋 추출
대상: `{레퍼런스URL}`
작업: 색·폰트·타입스케일 + 이미지(히어로 후보 1440폭↑) 인벤토리.
산출: design-tokens.json 초안 + 에셋표. 순백/순흑은 텍스트색 오추출 표기.
이걸로: :root/#nk-skinN CSS 변수 + 히어로 배경 확정.

### 27. 레퍼런스 구조 분석 (영역별)
대상: `{레퍼런스URL}`
작업: 헤더·히어로·상품그리드·배너·푸터를 영역별 분해(70% 영감+30% 변형).
산출: 영역별 구조표(레퍼/변형안) + 와이어 설명.
이걸로: 재설계 섹션 골격.

### 28. Figma → 토큰·컴포넌트
대상: `{Figma URL/node-id}`
작업: 색·타이포·간격·그림자 토큰 + variant/state별 컴포넌트. 전 페이지·프레임 나열·확인 후 순서대로.
산출: 토큰 JSON + 컴포넌트 스펙(Atomic). 하드코딩 금지.
이걸로: CSS 변수 + 컴포넌트 명세.

### 29. 디자인 요소 → 카페24 모듈 매핑
대상: `skin{N}/layout/basic/{main,layout}.html` + `index.html`
작업: 재설계 요소를 살아있는 모듈에 매핑(로고→LogoTop, GNB→category, 진열→product_listmain_N, 푸터→footer/Info/LogoBottom).
산출: "요소 → 모듈 → 파일:줄 → 바인딩변수({$...})" 표.
이걸로: 모듈 안 깨는 마크업 주입 좌표.

### 30. base 함정 일괄 차단안
대상: 25번 결과 + `skin-method-detect.md`
작업: 고정폭·float·가상요소선·테마색(theme01 #d0ac88)·!important를 `#nk-skinN`으로 차단. EZ면 late-load 대비 ID 스코프 + 사유주석 !important.
산출: `nk-ez-override.css` 차단 블록 + !important 사유.
이걸로: 구현 전 base 침범 선제 봉쇄.

### 31. HTML 네이티브 주입 (A안)
대상: `skin{N}/layout/basic/main.html` + `index.html`
작업: 재설계를 모듈 바인딩과 함께 주입. body id, 헤더/푸터 미니멀(모듈 보존), `_nk/css` 분리. `layout.html`(서브공통) 무수정.
산출: main·index·_nk/css. 서브페이지 영향 0.

### 32. EZ 선별이식 + strip (B안)

작업: 07/08로 선별이식(dangling 0)→재설계 주입→`data-ez` strip. EZST/Swiper 순서 점검.
산출: 이식 목록 + dangling 0 + strip 후 data-ez 0.
이걸로: EZ 경로 구현 + HTML-clean 수렴.

### 33. 3축 게이트 100점
대상: 미리보기 URL(`?skin=skin{N}`)
작업: visual(레퍼vs결과 PC+모바일 데스크톱UA)+override(high 0·dangling 0)+rule(blocking 0) 병렬. 미달 축 자가수정→재측정.
산출: 3축 리포트 + 캡처 증거. aggregate≥0.85·각≥0.70.
