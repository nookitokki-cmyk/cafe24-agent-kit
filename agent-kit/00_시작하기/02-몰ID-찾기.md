# 02. 몰 ID 찾기

## 방법 1 — 쇼핑몰 주소 (가장 쉬움)

브라우저 주소창:

```
https://여기가몰ID.cafe24.com/
```

**`여기가몰ID`** 가 몰 ID입니다.

예: `https://ecudemo400786.cafe24.com/` → 몰 ID = `ecudemo400786`

> ⚠️ 이 URL 규칙은 **관례**입니다. 공식 한 줄 정의는 AUDIT §A-1a 미확인.  
> ✅ API 호스트는 `{몰ID}.cafe24api.com` ([Admin OAuth](https://developers.cafe24.com/docs/api/admin/#oauth2)).

---

## 방법 2 — 관리자 화면

쇼핑몰 관리자 로그인 후 상단·설정에 표시되는 **쇼핑몰 아이디**를 확인합니다. (화면은 몰마다 다를 수 있음 ⚠️)

---

## 방법 3 — API로 교차 확인 (MCP)

OAuth·API 설정에 넣은 `mall_id`로 `cafe24_auth_status` 또는 `cli.py status`가 성공하면 **그 문자열이 맞는 몰 ID**입니다. ✅

---

## 흔한 실수

| 실수 | 올바른 것 |
|------|-----------|
| `cafe24.com` 전체를 몰 ID로 씀 | **앞부분 한 덩어리**만 |
| 파트너 계정 ID와 몰 ID 혼동 | 샘플몰 URL의 앞부분이 **몰 ID** (파트너 로그인 ID와 다를 수 있음 ⚠️) |
