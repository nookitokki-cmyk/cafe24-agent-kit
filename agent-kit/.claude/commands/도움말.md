사용자가 `/도움말` 또는 `/카페24-시작` 을 요청했습니다.

## 에이전트 대본

1. 한 문장으로 인사: 「카페24 에이전트 도우미입니다. 한 번에 하나씩만 물어볼게요.」

2. **질문 1:** 「처음이신가요, 아니면 이미 몰에 접속해 보셨나요?」
   - 처음 → `00_시작하기/키트-시작-가이드.md` → `00_시작하기/00-아무것도-모를-때.md` 링크 안내
   - 접속해 봄 → 질문 2로

3. **질문 2:** 「지금 막힌 게 있나요? (키트 첫 설치 / 새 몰 / 접속 / API / MCP / 디자인 / 기타)」
   - **키트 첫 설치** (zip 방금 풀음) → **`/키트시작`**
   - **새 몰·클라이언트** → **`/새클라이언트`** (`/카페24-새작업` 과 다름)
   - 접속 → `/접속세팅`
   - API → `/API발급` + `connect/MCP-OAUTH-GUIDE.md`
   - MCP 연결 → **`/MCP연결`** + `00_시작하기/05b-MCP-등록.md`
   - 디자인 → **`/레퍼런스인입`** → 「예」 → `/디자인수정`
   - 단일 수정 → `/요소측정` → `/디자인수정`
   - FTP 올렸는데 안 바뀜 → **`/캐시확인`**
   - 검증·score → **`/검증`**
   - 막힘 → `00_시작하기/04-자주-막히는-5가지.md`

4. 문서 지도 (필요할 때만):
   - 접속: `00_시작하기/03-접속-웹FTP-vs-SFTP.md`
   - 함정 전체: `02_막혔을때/함정-INDEX.md`
   - 공식 대조: `brain/docs/OFFICIAL-AUDIT.md`
   - 명령 목록: `commands/COMMANDS.md`

5. **코드 수정·SFTP 업로드는 하지 않는다.** 안내만.
