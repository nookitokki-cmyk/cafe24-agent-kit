- [ ] `editor_type` = **H** (HTML). E/D/W/C 이면 HTML 워크플로 자동 적용 금지 §D
- [ ] 백업·업로드 전 사용자 OK

**레퍼런스/시안 1:1:** 인입표 「예」 전 **HTML/CSS/FTP 금지** (`01_작업하기/workflows/05-reference-intake.md`).

## 시작 멘트

「실측 시트 기준으로 작업할게요. 먼저 `brain/docs/CAFE24-SMARTDESIGN-AGENT.md` 규칙을 따릅니다.」

## 작업 순서

1. `cafe24_read_page` 또는 SFTP로 **현재 파일** 읽기
2. `module=""` · `{$변수}` · `data-ez-*` **훼손 금지** (EZ는 사용자가 전환 요청 시만)
3. 변경은 `_nk/` · `custom.css` 우선
4. 업로드 전 변경 요약 + **사용자 컨펌**
5. PC + 모바일 검증 URL `?v=N`
