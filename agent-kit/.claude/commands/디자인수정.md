## 작업 순서

1. `cafe24_read_page` 또는 SFTP로 **현재 파일** 읽기
2. `module=""` · `{$변수}` · `data-ez-*` **훼손 금지** (EZ는 사용자가 전환 요청 시만)
3. 변경은 `_nk/` · `custom.css` 우선
4. 업로드 전 변경 요약 + **사용자 컨펌**
5. PC + 모바일 검증 URL `?v=N`
