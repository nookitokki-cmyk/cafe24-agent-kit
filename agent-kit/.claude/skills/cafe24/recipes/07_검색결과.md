# 검색결과 — 키워드 검색 결과 페이지

## 이게 뭔가요?
고객이 쇼핑몰 검색창에 "원피스", "파란 셔츠" 같은 단어를 입력했을 때 나오는 결과 페이지입니다.
검색어가 화면 상단에 크게 표시되고, 아래에 관련 상품들이 그리드로 나열됩니다.
검색된 상품이 없을 때는 "검색 결과가 없습니다" 메시지와 함께 추천 상품을 보여줄 수 있습니다.

## 어떤 페이지에서 쓰나요?
- 검색 결과 페이지: `search.html` (또는 `search_result.html`)
- 검색창에서 엔터를 누르거나 검색 버튼을 클릭하면 이 페이지로 이동합니다.

## 핵심 변수 미리 알기

> `{$keyword}` — 고객이 입력한 검색어를 자동으로 불러오는 변수입니다.
> HTML 어디서든 `{$keyword}`를 쓰면 "원피스", "청바지" 같은 실제 검색어가 출력됩니다.
>
> `{$search_count}` — 검색된 상품 총 개수를 불러오는 변수입니다.
> 결과가 0개일 때 "검색 결과 없음" 처리를 할 때 유용합니다.

## 필요한 모듈 (순서대로)

| 모듈 ID | 역할 | 필수여부 |
|---------|------|---------|
| `product_search` 또는 `search_result` | 검색 결과 상품 목록 | 필수 |
| `ModulePagination` | 하단 페이지 번호 버튼 | 필수 |
| `ModuleProductBest` 또는 `ModuleProductNew` | 검색 결과 없을 때 대체 상품 추천 | 선택 |

> **카페24 버전에 따라 모듈 이름이 다를 수 있습니다.**
> 스마트디자인 기본 스킨의 `search.html` 파일을 열어 기존에 쓰인 모듈 이름을 먼저 확인하세요.
> 아래 코드에서는 `product_search` 기준으로 작성했습니다.

## 복사해서 쓰는 HTML

```html
<!--@@--
페이지 타입: 검색 결과 (search.html)
레시피: 07_검색결과
--@@-->

<div class="nk-search-result-wrap">

  <!-- ① 검색어 표시 + 결과 수 -->
  <div class="nk-search-heading">
    <h1 class="nk-search-title">
      <!-- {$keyword}: 고객이 입력한 검색어 자동 출력 -->
      "<span class="nk-keyword">{$keyword}</span>"
      검색 결과
    </h1>
    <p class="nk-search-count">
      총 <strong>{$search_count}</strong>개의 상품을 찾았습니다.
    </p>
  </div>

  <!-- ② 정렬 선택 -->
  <div class="nk-search-toolbar">
    <select class="nk-sort-select" onchange="location.href=this.value">
      <option value="{$sort_url_date}" {$date_selected}>최신순</option>
      <option value="{$sort_url_best}" {$best_selected}>인기순</option>
      <option value="{$sort_url_price_asc}" {$price_asc_selected}>낮은가격순</option>
      <option value="{$sort_url_price_desc}" {$price_desc_selected}>높은가격순</option>
    </select>
  </div>

  <!-- ③ 검색 결과 상품 목록 -->
  <div class="nk-search-list">
    <module name="product_search" display_count="16" column="4">

      <!-- 검색 결과가 없을 때 -->
      <div module="empty">
        <div class="nk-no-result">
          <p class="nk-no-result-msg">
            "<strong>{$keyword}</strong>"에 대한 검색 결과가 없습니다.
          </p>
          <ul class="nk-search-tips">
            <li>검색어를 정확하게 입력했는지 확인해 주세요.</li>
            <li>검색어를 더 짧게 또는 다른 단어로 바꿔보세요.</li>
            <li>초성이나 띄어쓰기를 확인해 주세요.</li>
          </ul>
          <!-- 다시 검색 폼 -->
          <form class="nk-retry-search" method="get" action="/product/search.html">
            <input type="text" name="keyword" class="nk-form-input"
              placeholder="다시 검색해보세요" value="{$keyword}">
            <button type="submit" class="nk-search-submit-btn">검색</button>
          </form>
        </div>

        <!-- 결과 없을 때 추천 상품 (선택) -->
        <div class="nk-recommend-section">
          <h2 class="nk-recommend-title">이런 상품은 어떠세요?</h2>
          <div class="nk-product-grid">
            <module name="ModuleProductBest">
              <div class="nk-product-item">
                <a href="{$product_url}">
                  <div class="nk-product-thumb">
                    <img src="{$thumbnail_url}" alt="{$product_name}" loading="lazy">
                  </div>
                  <div class="nk-product-info">
                    <p class="nk-product-name">{$product_name}</p>
                    <p class="nk-price-sale">{$price|number_format}원</p>
                  </div>
                </a>
              </div>
            </module>
          </div>
        </div>
      </div>

      <!-- 검색 결과가 있을 때 -->
      <div module="list">
        <ul class="nk-product-grid">
          <li module="item" class="nk-product-item">
            <a href="{$product_url}">
              <!-- 상품 이미지 -->
              <div class="nk-product-thumb">
                <img src="{$thumbnail_url}" alt="{$product_name}" loading="lazy">
                <!-- 품절 표시 -->
                <span class="nk-badge-soldout" {$soldout|display}>품절</span>
              </div>
              <!-- 상품 정보 -->
              <div class="nk-product-info">
                <p class="nk-product-brand">{$brand_name}</p>
                <!-- 검색어가 포함된 상품 이름 — 검색어 부분이 강조 표시됨 -->
                <p class="nk-product-name">{$product_name}</p>
                <div class="nk-product-price">
                  <span class="nk-price-original {$sale_price|display}">
                    {$price|number_format}원
                  </span>
                  <span class="nk-price-sale">{$sale_price|number_format}원</span>
                </div>
              </div>
            </a>
          </li>
        </ul>
      </div>

    </module>
  </div>

  <!-- ④ 페이지 번호 -->
  <div class="nk-pagination">
    <module name="ModulePagination" />
  </div>

  <!-- ⑤ 다른 검색어로 다시 찾기 (결과가 있을 때도 하단에 표시) -->
  <div class="nk-search-again">
    <form method="get" action="/product/search.html" class="nk-search-bottom-form">
      <input type="text" name="keyword" class="nk-form-input"
        placeholder="다른 검색어로 찾기" value="{$keyword}">
      <button type="submit" class="nk-search-submit-btn">검색</button>
    </form>
  </div>

</div>
```

## 여기만 바꾸세요

| 바꿀 부분 | 현재 값 | 설명 |
|-----------|---------|------|
| `display_count` | `16` | 한 페이지에 보여줄 검색 결과 상품 수 |
| `column` | `4` | 한 줄에 몇 개씩 표시. CSS 열 수와 맞춰야 함 |
| 검색 팁 문구 | "검색어를 정확하게 입력..." | 원하는 안내 문구로 수정 가능 |
| `이런 상품은 어떠세요?` | 텍스트 | 결과 없을 때 추천 섹션 제목 |
| `nk-search-submit-btn` 배경색 | `#111` | 브랜드 컬러로 변경 |

> **검색 결과 페이지 URL은 어떻게 설정하나요?**
> 카페24에서 검색창(`<input type="text" name="keyword">`)의 `form action`을 `/product/search.html`로 지정하면 됩니다. 스킨 헤더의 검색 폼 action 속성을 확인하세요.

## `{$keyword}` 변수 활용 팁

```html
<!-- 검색어를 제목에 출력 -->
<h1>"{$keyword}" 검색 결과</h1>

<!-- 검색 후 다시 같은 검색어로 폼에 채워두기 (사용자 편의) -->
<input type="text" name="keyword" value="{$keyword}">

<!-- 검색 결과 수 표시 -->
<p>총 {$search_count}개</p>

<!-- 검색어가 없으면 특정 내용 숨기기 -->
<div {$keyword|display}>검색어가 있을 때만 보임</div>
```

## 자주 하는 실수

1. **검색 결과가 나오는데 `{$keyword}`가 화면에 그대로 텍스트로 표시돼요**
   → HTML 파일의 맨 위에 카페24 스마트디자인 지시어(`<!--@@--...--@@-->`)가 없거나 잘못된 경우입니다. 파일 맨 위의 `<!--@@--` 블록이 올바르게 있는지 확인하세요. 지시어가 없으면 변수를 인식하지 못합니다.

2. **검색 결과가 0개인데 "검색 결과 없음" 화면이 안 나오고 빈 화면만 나와요**
   → `<div module="empty">` 블록이 `<module name="product_search">` 태그 안에 있어야 합니다. 모듈 태그 밖으로 빠져나온 경우 카페24가 인식하지 못합니다. 코드 들여쓰기를 확인해 구조가 올바른지 검토하세요.
