# 게시판·공지사항 — 이벤트·공지·FAQ 게시판 만들기

## 이게 뭔가요?
공지사항, 이벤트, FAQ처럼 글 목록이 있고, 클릭하면 글 내용을 볼 수 있는 게시판 페이지입니다.
카페24에서 게시판은 숫자 번호로 구별됩니다.
예를 들어 공지사항이 1번 게시판, 이벤트가 2번 게시판, FAQ가 3번 게시판인 식입니다.
이 번호를 모듈 이름 뒤에 붙여서 사용하는 것이 카페24 게시판의 핵심 규칙입니다.

## 게시판 번호 개념 (비코더가 가장 헷갈리는 부분)

> **게시판 번호란?**
> 카페24 관리자 → [게시판] → [게시판 관리]에서 각 게시판에 자동으로 부여된 숫자입니다.
> 예를 들어 "공지사항" 게시판의 번호가 `3`이라면, 모든 모듈 이름 뒤에 `_3`을 붙입니다.
>
> | 게시판 이름 | 번호 | 사용하는 모듈 예시 |
> |------------|------|--------------------|
> | 공지사항 | 3 | `board_list_3`, `board_notice_3` |
> | 이벤트 | 4 | `board_list_4`, `board_notice_4` |
> | FAQ | 5 | `board_list_5`, `board_notice_5` |
>
> **이 번호는 내 쇼핑몰마다 다릅니다.** 관리자에서 꼭 직접 확인하세요.
> 아래 코드의 `{no}` 자리에 실제 번호를 넣으면 됩니다.

## 어떤 페이지에서 쓰나요?
- 게시판 목록 페이지: `board_list.html`
- 게시판 글 보기 페이지: `board_read.html`
공지사항, 이벤트, FAQ 모두 같은 파일을 씁니다 (게시판 번호로 구분).

## 필요한 모듈 (순서대로)

| 모듈 ID | 역할 | 필수여부 |
|---------|------|---------|
| `board_title_{no}` | 게시판 이름(타이틀) 표시 | 필수 |
| `board_notice_{no}` | 공지글(항상 위에 고정된 글) 목록 | 선택 |
| `board_listheader_{no}` | 목록 헤더 (번호·제목·작성일 등 컬럼명) | 선택 |
| `board_list_{no}` | 일반 글 목록 | 필수 |
| `board_paging_{no}` | 페이지 번호 버튼 | 필수 |
| `board_search_{no}` | 검색창 | 선택 |

> `{no}` 자리에 실제 게시판 번호를 넣으세요.
> 예: 게시판 번호가 3이면 → `board_list_3`, `board_paging_3`

## 복사해서 쓰는 HTML

### [A] 게시판 목록 페이지 (`board_list.html`)

```html
<!--@@--
페이지 타입: 게시판 목록 (board_list.html)
레시피: 05_게시판-공지사항
게시판 번호: {no} ← 관리자에서 확인 후 숫자로 교체
--@@-->

<div class="nk-board-wrap">

  <!-- ① 게시판 이름 -->
  <module name="board_title_{no}">
    <h1 class="nk-board-title">{$board_name}</h1>
  </module>

  <!-- ② 검색창 -->
  <div class="nk-board-search">
    <module name="board_search_{no}">
      <form method="get" class="nk-search-form">
        <select name="search_field" class="nk-search-select">
          <option value="subject">제목</option>
          <option value="content">내용</option>
          <option value="writer">작성자</option>
        </select>
        <input type="text" name="search_word" value="{$search_word}"
          class="nk-search-input" placeholder="검색어를 입력하세요">
        <button type="submit" class="nk-search-btn">검색</button>
      </form>
    </module>
  </div>

  <!-- ③ 공지글 목록 (항상 최상단 고정) -->
  <div class="nk-board-notice-area">
    <module name="board_notice_{no}">
      <ul class="nk-board-list">
        <li module="item" class="nk-board-item nk-board-notice">
          <span class="nk-board-badge">공지</span>
          <a href="{$read_url}" class="nk-board-subject">{$subject}</a>
          <span class="nk-board-date">{$write_date}</span>
        </li>
      </ul>
      <!-- 공지글 없을 때는 아무것도 안 나옴 (정상) -->
    </module>
  </div>

  <!-- ④ 일반 글 목록 -->
  <div class="nk-board-list-area">
    <module name="board_list_{no}">
      <!-- 글이 없을 때 -->
      <div module="empty">
        <p class="nk-empty-msg">등록된 게시물이 없습니다.</p>
      </div>
      <!-- 글이 있을 때 -->
      <div module="list">
        <!-- 목록 헤더 -->
        <div class="nk-board-list-header">
          <span class="nk-col-num">번호</span>
          <span class="nk-col-subject">제목</span>
          <span class="nk-col-writer">작성자</span>
          <span class="nk-col-date">날짜</span>
          <span class="nk-col-hit">조회</span>
        </div>
        <!-- 개별 글 행 -->
        <ul class="nk-board-list">
          <li module="item" class="nk-board-item">
            <!-- 글 번호 -->
            <span class="nk-col-num">{$no}</span>
            <!-- 제목 + 댓글 수 -->
            <div class="nk-col-subject">
              <a href="{$read_url}" class="nk-board-subject">
                {$subject}
                <!-- 첨부파일 있을 때 아이콘 -->
                <span class="nk-attach-icon {$attach|display}">📎</span>
              </a>
              <!-- 댓글이 있을 때 [댓글수] 표시 -->
              <span class="nk-reply-count {$reply_count|display}">[{$reply_count}]</span>
            </div>
            <!-- 작성자 -->
            <span class="nk-col-writer">{$writer}</span>
            <!-- 날짜 -->
            <span class="nk-col-date">{$write_date}</span>
            <!-- 조회수 -->
            <span class="nk-col-hit">{$hit}</span>
          </li>
        </ul>
      </div>
    </module>
  </div>

  <!-- ⑤ 글쓰기 버튼 (회원만 보이도록 하려면 관리자 설정에서 제어) -->
  <div class="nk-board-actions">
    <a href="{$write_url}" class="nk-btn-write">글쓰기</a>
  </div>

  <!-- ⑥ 페이지 번호 -->
  <div class="nk-board-paging">
    <module name="board_paging_{no}" />
  </div>

</div>
```

### [B] 게시판 글 보기 페이지 (`board_read.html`)

```html
<!--@@--
페이지 타입: 게시판 글 보기 (board_read.html)
레시피: 05_게시판-공지사항 (글 보기)
게시판 번호: {no}
--@@-->

<div class="nk-board-read-wrap">

  <!-- 글 제목 + 메타 -->
  <div class="nk-read-header">
    <h1 class="nk-read-subject">{$subject}</h1>
    <div class="nk-read-meta">
      <span class="nk-read-writer">{$writer}</span>
      <span class="nk-read-date">{$write_date}</span>
      <span class="nk-read-hit">조회 {$hit}</span>
    </div>
  </div>

  <!-- 글 본문 -->
  <div class="nk-read-content">
    {$content}
  </div>

  <!-- 첨부파일 (있을 때만) -->
  <div class="nk-read-attach {$attach|display}">
    <strong>첨부파일:</strong> {$attach_file}
  </div>

  <!-- 이전글 / 다음글 -->
  <div class="nk-read-nav">
    <div class="nk-read-nav-item">
      <span class="nk-nav-label">이전글</span>
      <a href="{$prev_url}" class="nk-nav-link">{$prev_subject}</a>
    </div>
    <div class="nk-read-nav-item">
      <span class="nk-nav-label">다음글</span>
      <a href="{$next_url}" class="nk-nav-link">{$next_subject}</a>
    </div>
  </div>

  <!-- 목록으로 / 수정 / 삭제 버튼 -->
  <div class="nk-read-actions">
    <a href="{$list_url}" class="nk-btn-list">목록으로</a>
    <a href="{$modify_url}" class="nk-btn-modify {$modify|display}">수정</a>
    <button type="button" class="nk-btn-delete {$delete|display}"
      onclick="deleteBoard('{$no}')">삭제</button>
  </div>

</div>
```

## 여기만 바꾸세요

| 바꿀 부분 | 현재 값 | 설명 |
|-----------|---------|------|
| 모든 `{no}` | `{no}` | 관리자에서 확인한 실제 게시판 번호로 교체 |
| 목록 헤더 텍스트 | `번호`, `제목`, `작성자` 등 | 원하는 이름으로 변경 |
| 검색 항목 | `제목`, `내용`, `작성자` | 필요한 항목만 남기기 |

> **게시판 번호 확인 방법**
> 카페24 관리자 → [게시판] → [게시판 관리] → 각 게시판 이름 옆에 번호가 표시됩니다.
> 이 번호가 3이면 `board_list_{no}` → `board_list_3`으로 교체합니다.

## 자주 하는 실수

1. **`{no}` 자리를 그대로 두고 저장했더니 글 목록이 안 나와요**
   → `{no}`는 반드시 실제 게시판 번호(예: 3)로 교체해야 합니다. 모듈 이름 안에 그대로 남아있으면 카페24가 인식하지 못합니다. 파일 전체에서 `{no}`를 검색해 모두 숫자로 바꾸세요.

2. **공지글 모듈을 넣었는데 아무것도 안 나와요**
   → `board_notice_{no}` 모듈은 관리자에서 글을 "공지글로 설정"해야 나타납니다.
   글 작성/수정 화면에서 "공지글 여부" 체크박스를 체크하세요.
