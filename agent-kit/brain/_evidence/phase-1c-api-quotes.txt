# Phase 1-C — 스마트디자인 공식 가이드 인용 (2026-06-19)

## 출처 A: 디자인 기본 이해하기
URL: https://developers.cafe24.com/design/front/smart/sdsupport/basic
(HTML: docs/_evidence/sds_developers_cafe24_com_design_front_smart_sdsupport_basic.html)

[@layout 지시어]
- 레이아웃 HTML 맨 위: <!--@layout(layout.html)--> 형태
- 마우스오버로 레이아웃 파일 열기 가능

[모듈 Module]
- module="모듈아이디" 에 따라 판단·구동
- HTML + 변수 조합이 최소 단위
- 예: module="member_join", product_listmain_1

[변수 Variable]
- {$ ... } 사이에 값 입력
- 변수는 모듈에 상속되어 동작, 모듈마다 사용 가능 변수가 정해짐
- 예: {$image_medium}, {$image_small}, {$image_large}

[반복 횟수 — 주석변수]
- 우선순위: $only_html > $count > DB 카운트 설정
- 동일 항목 반복 vs 마지막 블록만 반복 — 케이스별 예시 있음

[변수修飾 — 공식 예]
- {$product_name|cut:10,...}
- {$name_or_img_tag|imgconv:'이미지 주소'}
- {$new_icon|strconv:신상품}
- {$notice_icon|replace:공지,공지사항}
