## 🚶 작업 흐름

### 1️⃣ 이미지 6장 준비 (PC 3 + 모바일 3)

로컬에:
```
03_references/images/
  ├── banner-pc-1.jpg
  ├── banner-pc-2.jpg
  ├── banner-pc-3.jpg
  ├── banner-mo-1.jpg
  ├── banner-mo-2.jpg
  └── banner-mo-3.jpg
```

### 2️⃣ 배너 카피 정리 (텍스트 파일로)

`01_요청사항/배너_카피_v1.md` 생성:
```markdown
# 메인 배너 카피 (3슬라이드)

## 슬라이드 1
- 헤드: 2026 신규 컬렉션
- 부제: 지금 만나보세요
- 버튼: SHOP NOW
- 링크: /product/list.html?cate_no=43

## 슬라이드 2
- 헤드: 첫 구매 30% 할인
- 부제: 회원가입 시 즉시 사용
- 버튼: 회원가입
- 링크: /member/join.html

## 슬라이드 3
- 헤드: 무료 배송 이벤트
- 부제: 5만원 이상 구매
- 버튼: 자세히
- 링크: /board/free/list.html
```

### 3️⃣ AI 에게 한 줄 지시

```
메인 배너 3슬라이드를 통째로 교체해줘.

- skin: skin7
- 이미지: 03_references/images/banner-pc-1.jpg ~ pc-3.jpg
- 모바일 이미지: banner-mo-1.jpg ~ mo-3.jpg
- 카피: 01_요청사항/배너_카피_v1.md
- 모렌비 슬라이더 쓰는지 자동 슬라이더인지 확인 후 진행
- PC 1920×600, 모바일 750×900 비율 유지
- 자동 재생, 5초 간격
- 좌우 화살표 + 아래 점 인디케이터
```

### 4️⃣ AI 가 하는 일

```
1. 현재 메인 배너 시스템 파악:
   - 모렌비 배너관리자? 또는 _nk/inc/main-hero.html 직접?
   - admin 에서 등록하는 방식인지 코드에 박힌 건지

2. 이미지 6장 SFTP 업로드 → /skin7/_nk/img/banner/

3. HTML 작성 (Swiper.js 슬라이더 + 카페24 카피)

4. CSS:
   .nk-main-hero {
     width: 100%;
     aspect-ratio: 1920/600;
   }
   @media (max-width: 768px) {
     .nk-main-hero { aspect-ratio: 750/900; }
   }

5. PC·모바일 다른 이미지 source 분기

6. 검증 가이드: PC 1920px·모바일 360px·아이패드 768px 셋 다 확인
```

### 5️⃣ 검증

```
https://{본인몰}.cafe24.com/?v=20
```

체크리스트:
- [ ] PC 에서 슬라이드 3장 자동 재생 ✓
- [ ] 5초마다 다음 슬라이드 ✓
- [ ] 좌우 화살표 클릭 OK ✓
- [ ] 아래 점 인디케이터 OK ✓
- [ ] 버튼 클릭 시 지정 URL 로 이동 ✓
- [ ] 모바일에서 세로 이미지 ✓
- [ ] 모바일 터치 스와이프 OK ✓
- [ ] 첫 로딩 시 깜빡임 없음 ✓

## ⚠️ 자주 막히는 곳

### 슬라이드가 세로로 쌓임 (가로 캐러셀 안 됨)
→ 함정 #F1 (메인진열 복제 견본 ≥2개 규칙 + @css/@js base fallback)
```
/카페24-도와줘
메인 배너가 가로 슬라이드 안 되고 세로로 쌓여.
```

### 모바일에서 PC 이미지가 나옴
→ 모바일 skin 따로 작업 필요
```
모바일에서 PC 배너 이미지 나옴.
모바일 skin (m_skin 또는 mobile) 도 같이 처리해줘.
```

### admin 에서 등록한 배너가 안 보임
→ 카페24 admin 의 모렌비 / 배너관리자 의 표시 여부 OFF 상태일 수 있음
```
admin → 디자인 → 배너관리에서 배너 ON 상태 확인 필요.
```

## 💡 응용

### A. 슬라이드별 다른 효과
```
슬라이드 1 은 fade-in, 슬라이드 2 는 슬라이드, 슬라이드 3 은 zoom-in
```

### B. 버튼 hover 시 색 변환
```
버튼 hover 시 배경색 검정 → 흰색, 글자 흰색 → 검정으로 반전.
```

```
오전 9시~18시는 슬라이드 1, 그 외는 슬라이드 2 메인으로.
JS 로 처리.
```
