# 시나리오 01 — 헤더 로고 교체 🟢

> 범용 시나리오 — 카페24 + AI 에이전트 작업자용

> **목표**: 카페24 헤더의 로고 이미지를 새 이미지로 교체. 20분.
> 02_첫_작업 보다 한 단계 깊은 실습. 이미지 업로드·경로 지정까지 포함.

---

## 🎯 이번 시나리오로 배우는 것

- ✅ 이미지 파일을 카페24 서버로 SFTP 업로드
- ✅ HTML 안 `<img src=...>` 경로 수정
- ✅ 로고 크기·간격 조정 (CSS)
- ✅ PC·모바일 다른 로고 쓰기 (선택)

---

## 🔧 준비물

| 항목 | 확인 |
|---|---|
| 새 로고 PNG 파일 (투명 배경 권장) | 가로 240×60px 안팎 |
| (선택) 모바일용 작은 로고 | 가로 120×40px 안팎 |
| `clients/{본인}/cafe24/` 폴더 열려 있음 | OK |
| skin 번호 | OK |

> 로고가 없으면 샘플 자료 데모 로고 사용 가능.

---

## 🚶 작업 흐름

### 1️⃣ 로고 파일 준비

로컬에:
```
clients/{본인}/cafe24/03_references/images/logo-new.png
```

위치에 새 로고 저장. 디자인 도구 가이드에서 Figma 로 만드는 법 참고.

### 2️⃣ AI 에게 한 줄 지시

```
clients/{본인}/cafe24/03_references/images/logo-new.png 를
헤더 로고로 교체해줘.

- skin: skin7
- 현재 로고 위치 파악
- SFTP 로 카페24 서버에 logo-new.png 업로드
- 헤더 HTML의 <img src=...> 경로 변경
- 로고 크기는 가로 200px 정도로 조정
- PC + 모바일 둘 다 적용
```

### 3️⃣ AI 가 하는 일

```
1. 현재 헤더 찾기 — _nk/inc/header.html 또는 module 안 <img>
2. logo-new.png 를 /skin7/_nk/img/ 로 SFTP 업로드
3. HTML 의 <img src="..."> 경로 변경:
   <img src="/skin7/_nk/img/logo-new.png" alt="로고">
4. CSS 로 크기 조정:
   .nk-header__logo img {
     width: 200px;
     height: auto;
   }
5. 모바일도 같이 (또는 다른 로고로):
   @media (max-width: 768px) {
     .nk-header__logo img { width: 140px; }
   }
```

### 4️⃣ 검증

브라우저:
```
https://{본인몰}.cafe24.com/?v=10
```

체크리스트:
- [ ] PC 헤더에 새 로고 ✓
- [ ] 모바일(360px)에서도 새 로고, 적절한 크기 ✓
- [ ] 클릭 시 메인으로 이동 ✓ (링크 안 끊겼는지)
- [ ] 로고 위·아래 간격 자연스러움 ✓

## 💡 응용

### A. PC·모바일에 다른 로고
```
PC 에는 logo-wide.png (가로형, 240×60)
모바일에는 logo-square.png (정사각, 60×60)
```

### B. 호버 시 로고 색 바꾸기
```
헤더 로고에 hover 효과 — opacity 0.7 로 살짝 흐려지게.
```

### C. 스크롤 시 로고 작아지기 (고급)
```
스크롤 100px 이상 내려가면 로고 크기가 200 → 140 으로 줄어드는 sticky 효과.
```

C 는 🟡 중급. 실패해도 PC·모바일 기본 로고는 안 깨짐.

---

## ⚠️ 자주 막히는 곳

### 로고 안 보임
→ SFTP 업로드가 안 됐거나 경로가 틀림. 다음 입력:
```
/카페24-도와줘
새 로고 경로가 라이브에서 404 나는데 어디 봐야 해?
```

### 로고 위치가 깨짐
→ 기존 헤더의 flex / grid 가 새 이미지 비율에 안 맞음. AI 한테:
```
로고 크기 바꿨더니 헤더 메뉴랑 겹쳐.
헤더 flex 정렬도 같이 봐줘.
```

### 모바일만 로고 안 보임
→ 모바일 skin (별도 폴더) 도 같이 수정해야 함. AI 가 자동으로 둘 다 처리하지만 명시:
```
PC skin7 만 됐어. 모바일 skin (보통 mobile 또는 m_skin) 도 같이 작업해줘.
```
